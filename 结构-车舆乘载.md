# 车「4」
* 贲初九：贲其趾，舍车而徒
* 困九四：来徐徐，困于金车，吝，有终
* 睽上九：睽孤，见豕负涂，载鬼一车，先张之弧，后说之弧，匪寇婚媾，往遇雨则吉
* 大有九二：大车以载，有攸往，无咎
# 舆「7」
* 师六三：师或舆尸，凶
* 师六五：田有禽，利执言，无咎。长子帅师，弟子舆尸，贞凶
* 剥上九：硕果不食，君子得舆，小人剥庐
* 睽六三：见舆曳，其牛掣，其人天且劓，无初有终
* 小畜九三：舆说辐，夫妻反目
* 大畜九二：舆说輹
* 大壮九四：贞吉悔亡，藩决不羸，壮于大舆之輹
# 乘「5」
* 屯六二：屯如邅如，乘马班如。匪寇婚媾，女子贞不字，十年乃字
* 屯六四：乘马班如，求婚媾，往吉，无不利
* 屯上六：乘马班如，泣血涟如
* 解六三：负且乘，致寇至，贞吝
* 同人九四：乘其墉，弗克攻，吉
# 载「3」

* 大有九二：大车以载，有攸往，无咎
